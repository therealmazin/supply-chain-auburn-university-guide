# Contact

## About the Author

## Get Help

### Technical Issues

If you're having trouble with:
- Installing Claude Code
- Running the prompts
- Getting different results

**First try:**
1. Check the [Troubleshooting](/docs/getting-started/install-claude-code#troubleshooting) section
2. Ask Claude Code itself for help (describe your error)
3. Search the [Anthropic documentation](https://docs.anthropic.com/)
